A song of nodes and edges: Network analysis in Game of Thrones

Data source: GitHub

To demonstrate the concept of network analysis, I built an interactive, force-directed graph of character relationships for the first novel of George R.R Martin's Game of Thrones series using D3.js. The Network of Thrones article by Andrew Beveridge and Jie Shan inspired the work. The source data gathered from the novels can be found on Andrew Beveridge's GitHub.

The characters that feature in the novel are listed as the nodes in the network. A connection or 'edge' is counted when two character's names appear within 15 words of each other.
