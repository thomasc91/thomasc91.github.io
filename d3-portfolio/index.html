About

The D3.js Portfolio is a collection of data visualisations created by Data Skunkworks.

D3.js is a JavaScript library for producing dynamic, interactive data visualisations in web browsers. D3.js gives us almost unlimited potentital for data visualisation, and is not tied to a proprietary framework or BI tool.

Each visualisation in the portfolio explores a different dataset and method of visualisation. Most of the charts interactive, allowing users to dig deeper into the data and uncover new insights.
